FakeStore E-Commerce React App

A responsive React-based e-commerce interface that interacts with the [FakeStoreAPI](https://fakestoreapi.com/). 
Users can browse products, view details, add/edit/delete products (simulated), and manage a shopping cart.

Features

- Browse a list of products from FakeStoreAPI
- View detailed product information
- Add new products
- Edit existing products
- Delete products (simulated)
- Add/remove products to/from cart
- Responsive design using Bootstrap
- Loading indicators and error handling
- Custom 404 Not Found page with redirect

Built With

- [React](https://reactjs.org/)
- [React Router](https://reactrouter.com/)
- [Bootstrap](https://getbootstrap.com/)
- [Axios](https://axios-http.com/)
- [FakeStoreAPI](https://fakestoreapi.com/)
